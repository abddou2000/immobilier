# Marrakech Real Estate Frontend

Frontend premium Next.js pour une agence immobiliere a Marrakech.

## Lancer le projet

```bash
npm install
npm run dev
```

Puis ouvrir :

```text
http://localhost:6895
```

## Routes

- `/`
- `/acheter`
- `/louer`
- `/quartiers`
- `/guides`
- `/agence`
- `/contact`

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- lucide-react
- Three.js
